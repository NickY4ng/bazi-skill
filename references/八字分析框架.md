## 一、理论基础（统一标准）

### 核心原则（《子平真诠》）

> "八字用神专凭月令，月无用神，徐寻格局"

1. **月令为纲**：月令是判断用神的核心出发点

3. **扶抑为用**：身强抑之，身弱扶之
4. **调候为助**：寒暖燥湿需调候（穷通宝鉴补充）

## 八，大运流年

### 大运排法

| 性别 | 年干 | 排法 |
|------|------|------|
| 男 | 阳年（甲丙戊庚壬） | 顺排 |
| 男 | 阴年（乙丁己辛癸） | 逆排 |
| 女 | 阳年（甲丙戊庚壬） | 逆排 |
| 女 | 阴年（乙丁己辛癸） | 顺排 |

### 起运年龄（yshicalculator通用算法）
```
起运年龄 = 出生到「下一个12节」天数 ÷ 3
```

**12节**：小寒、立春、惊蛰、清明、立夏、芒种、小暑、立秋、白露、寒露、立冬、大雪

**找节算法**：
1. 调API获取生日当天的下一个节气
2. 判断是"节"还是"气"（24节气中奇数位=节，偶数位=气）
3. 如果是"气"，找下一个"节"
4. 计算天数÷3，不足整月→+1岁

**大运干支公式**（序号i从1开始）：
```
顺排：第i步 = (月干/支index + i + 1) % 10/12
逆排：第i步 = (月干/支index - i - 1 + 10/12) % 10/12
```

**示例**：丁丑月（丁=3, 丑=1）→ 顺排第1步：戊寅 ✅

### 核心原则
```
命局 → 大运（10年趋势） → 流年（当年触发） → 吉凶
```

## 十一、专项分析维度（六大维度）

> 核心口诀：**穷通寿夭、妻财子禄**
> 分析维度：事业、财运、婚姻、六亲、健康、子女

### 1. 事业分析

| 组合 | 事业象义 |
|------|----------|
| 官印相生 | 仕途顺利、有靠山、适合公职 |
| 杀印相生 | 有魄力、能掌权、适合管理 |
| 伤官见官 | 易犯官非、不服管、创意路线 |
| 比劫重重 | 事业竞争大、易犯小人 |
| 食神生财 | 靠技能/创意吃饭 |

**事业层次**：用神得力+格局纯粹=事业成就高；用神无力+破格=事业阻滞

---

### 2. 财运分析

| 组合 | 财运象义 |
|------|----------|
| 身强财旺 | 富命，财来财聚 |
| 身弱财旺 | 富屋贫人，财多身不能担 |
| 食神生财 | 技能生财，稳健 |
| 偏财旺透 | 投资运、意外财、浮动收入 |
| 比劫夺财 | 财来财去、破财、三人共分 |

**财运层次**：身担+护财（食神或官杀护财）=富；身弱无根=贫

---

### 3. 婚姻分析（男看财星，女看官星）

| 男命 | 女命 |
|------|------|
| 正财为妻（正缘） | 正官为夫（正缘） |
| 偏财为妾/外遇 | 七杀为烂桃花 |
| 财星在日支为夫妻宫 | 官星在日支为夫妻宫 |

| 婚姻状态 | 命局特征 |
|----------|----------|
| 早婚 | 财官早透不被合冲 |
| 晚婚 | 财官不透或被合冲 |
| 婚变 | 财星被比劫冲；官杀混杂 |

### 4. 六亲分析

| 六亲 | 代表星 | 核心判断 |
|------|--------|----------|
