# CHANGELOG

All notable changes to this project will be documented in this file.

## [v1.0.0] - 2026-04-17

### 首发版本

**功能**
- 四柱排盘（年柱、月柱、日柱、时柱）
- 纳音五行
- 真太阳时校正（百度地图AK可选）
- 大运计算（支持顺排/逆排）
- 流年、流月、流日、流时（黄历API）
- 神煞查询（天乙贵人、文昌、桃花、驿马、羊刃等）
- 空亡查询
- 长生十二宫
- 藏干查询

**技术**
- 黄历API（必选）、百度地图AK（可选）
- 全动态数据，无硬编码业务逻辑
- 环境变量配置，无硬编码密钥
