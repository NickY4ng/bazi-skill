# Bazi Master - 八字命理分析工具

> 本Skill以**《子平真诠》**为理论核心，结合现代实用体系，统一标准化分析。
> **全链路计算**：输入生日（阳历/阴历均可）→ 自动完成四柱+大运+流年+流月+流日全链路计算。

---

## 输入输出规范

### 输入格式
```
生日：YYYY-MM-DD HH:MM [城市/地址]
性别：男/女
阴历/阳历：（未说明时需询问用户）
```
示例1（阳历）：`1995-01-11 14:30 北京`
示例2（阴历）：`1989-09-24 05:30 山东济宁`

**必问确认项**（用户未主动说明时必须问）：
1. **阴历还是阳历**——用户只给日期时必须确认
2. **性别**——男/女（决定大运顺逆）
3. **出生地**——用于真太阳时校正（县级以上即可，如"黑龙江佳木斯"）

**真太阳时校正**：出生地经度影响日/时柱分界，有地址时自动启用百度地理编码校正。
- 经度 120°E = 北京标准时，偏差 1° = ±4分钟
- 例：黑龙江(130°E) → 出生时间+40分钟 → 影响时柱和日柱分界
- 同时加入时差方程（天文时差，±16分钟季节波动）

**阴历支持**：自动调用 lunardate 库将阴历转为阳历后查询黄历API。
- 注意：lunardate库支持年份范围为 1900-2100 年

**乾造/坤造**：
- **乾造**：男命八字
- **坤造**：女命八字

### 输出：完整命盘
调用黄历API自动获取：
- 四柱干支（年柱、月柱、日柱、时柱）
- 四柱纳音（年纳音、月纳音、日纳音、时纳音）
- 节气信息（上一个节气、下一个节气及精确时间）
- 农历日期

## 第一步：四柱排盘（核心算法）

### 1.1 调用黄历API + 真太阳时校正
**步骤**：
1. 地理编码：用百度API将地址转为经纬度
   - AK: `iYvw4ERKmMyqp4xTdd0TS4SZSJIQICJy`
   - `GET https://api.map.baidu.com/geocoding/v3/?address=地址&ak=xxx&output=json`
   - 返回 `location.lng/lat`（经纬度）
2. 真太阳时校正：`校正分钟 = (经度 - 120) × 4`
3. 查黄历API：`GET https://api.t1qq.com/api/tool/day/time?key=...`
4. 时柱校正：根据校正后的真太阳时，重新查时柱干支
   - 时柱天干起法（五虎遁起时）：
     - 甲/己日起甲子，乙/庚日起丙子，丙/辛日起戊子，丁/壬日起庚子，戊/癸日起壬子
   - 时柱地支：23-1点=子时，1-3=丑时，...21-23=亥时
| `jieQi.prev/next` | 上一个/下一个节气及精确时间 |
| `lunarYear/Month/Day` | 农历日期 |

### 1.2 年柱排法（按节气，非元旦）
**规则**：年柱以**立春**为分界线，非1月1日。
- 若生日在当年立春之前，年柱用上一年
- 若生日在当年立春之后（含立春当天），年柱用当年

**公式**（用于校验或无API时）：
```
stemIndex = (公历年份 - 4) % 10
zhiIndex = (公历年份 - 4) % 12
年柱 = 天干[stemIndex] + 地支[zhiIndex]
```

**天干顺序**（index 0-9）：甲、乙、丙、丁、戊、己、庚、辛、壬、癸
**地支顺序**（index 0-11）：子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥

**示例**：1995-01-11，小寒后立春前（节气：1995-01-06小寒，1995-02-04立春）
- 生日在立春前 → 年柱取1994年：`(1995-4)%10=1→甲`，`(1995-4)%12=3→戌` → **甲戌**

### 1.3 月柱排法（按节气 + 五虎遁）

**规则**：月柱以每月**节**为分界（如立春、惊蛰、清明…），非每月1日。

**五虎遁公式**（月干计算）：
| 年干 | 正月天干 | 口诀 |
|------|----------|------|
| 甲 | 丙寅 | 甲己之年丙作首 |
| 乙 | 戊寅 | 乙庚之年戊为头 |
| 丙 | 庚寅 | 丙辛必定庚初起 |
| 丁 | 壬寅 | 丁壬壬位顺行流 |
| 戊 | 甲寅 | 戊癸之年何方发 |
| 己 | 丙寅 | 甲己之年丙作首 |
| 庚 | 戊寅 | 乙庚之年戊为头 |
| 辛 | 庚寅 | 丙辛必定庚初起 |
| 壬 | 壬寅 | 丁壬壬位顺行流 |
| 癸 | 甲寅 | 戊癸之年何方发 |

**月干计算公式**：
```
五虎遁起点 = {
  甲年→0（丙），乙年→2（戊），丙年→4（庚），
  丁年→6（壬），戊年→8（甲），己年→0（丙），
  庚年→2（戊），辛年→4（庚），壬年→6（壬），癸年→8（甲）
}
月干index = (五虎遁起点 + 月支index) % 10
月支index：正月=2（寅），二月=3（卯），…，十二月=1（丑）
```

**简化**：直接用API的 `GanZhiMonth`，以上公式用于理解原理。

### 1.4 日柱排法
**直接使用API的 `GanZhiDay`**，不自行计算。

### 1.5 时柱排法（5分钟=1刻，一时辰=8刻）
**直接使用API的 `GanZhiHour`**。

**说明**：

1. 优先看月支藏干是否有天干透出（透出为用）

3. 本气被冲克时，可参看中气

- 丑藏：己（本气）、癸（中气）、辛（余气）
- 均未透出 → 取本气 **己土** 为月令用神

## 第七步：自坐判断

> 自坐 = 日干在日支的十二长生位（即日干在地支的十二长生状态）。反映日主坐下的根基强弱。

**查法**：直接查"十二长生表"，以日干对应日支的位置。

**自坐十神判断方法**（正确方式）：
自坐的十神不是从十二长生状态直接映射，而是看**日支藏干与日干的生克关系**。

**步骤**：
1. 找出日支的藏干（本气、中气、余气）
2. 判断每个藏干与日干的生克关系
3. 根据同性/异性判定十神

**十神判别规则**：
| 关系 | 同性（阳干对阳支/阴干对阴支） | 异性（阳干对阴支/阴干对阳支） |
|------|----------------------------------|----------------------------------|
| 我生者 | 食神 | 伤官 |
| 我克者 | 偏财 | 正财 |
| 克我者 | 七杀 | 正官 |
| 生我者 | 偏印 | 正印 |
| 同我者 | 比肩 | 劫财 |

自坐本气的能量最大，自坐中气次之，余气最弱。

**验证示例**：1995-01-11，壬寅日柱
- 日干=壬，日支=寅
- 壬在寅 = 病（查十二长生表）→ 自坐**病**
- 寅藏：甲（食神）、丙（偏财）、戊（正财）
- 自坐食神生财格，坐下有根，身弱但有根气

## 第九步：大运计算（dayun_api.py 黄历API版）

### 9.1 核心逻辑（2026-04-17更新）
```
天数差 ÷ 3 = qiyun_age（岁）
阴历11月/12月出生 → 不+1
其他月份 → +1
起运日期 = 出生日期 + (qiyun_age × 365天)
每步大运 = 起运日期 + n×10年
```

### 9.2 算法（黄历API版）

**实现文件**：`dayun_api.py`

**数据源**：黄历API `https://api.t1qq.com/api/tool/day/time`

**12节集合**：立春、惊蛰、清明、立夏、芒种、小暑、立秋、白露、寒露、立冬、大雪、小寒

**顺排（find_next_jie）**：
1. 调用黄历API获取当前日期的节气信息（`jieQi.next`）
2. 如果是12节之一 → 直接返回节气时间和名称
3. 如果不是12节（是气）→ 以该节气时间为起点+1天，继续查下一个节气，递归最多30次

**逆排（find_prev_jie）**：
1. 调用黄历API获取当前日期的节气信息（`jieQi.prev`）
2. 如果是12节之一 → 直接返回
3. 如果不是12节 → 以该节气时间-1天，继续查上一个节气，递归最多30次

**顺逆排判断（同性相从）**：
- 阳年男 → 顺排
- 阴年女 → 顺排
- 阳年女 → 逆排
- 阴年男 → 逆排

**年柱阴阳判断**：天干index % 2 == 0 → 阳干

### 9.3 起运年龄计算（calc_start_age）

**核心公式（2026-04-17修正版）**：
```
天数差 ÷ 3 = qiyun_age（岁）
阴历11/12月 → start_age = floor(qiyun_age)
其他月份 → start_age = floor(qiyun_age) + 1
起运日期 = 出生日期时间 + (qiyun_age × 365)天
```

**年龄标注规则（测测APP风格）**：
- 小运区间：1岁 ~ 起运年龄
- 第一步大运区间：起运年龄 ~ (起运年龄+9)岁

**示例**：1997-10-05 女 02:30（顺排）
- 黄历API查到：下一个节气是寒露@1997-10-08 14:05
- 精确天数（考虑出生时间02:30）：**3.59天**
- qiyun_age = 3.59 ÷ 3 = **1.20岁**
- 非阴历11/12月 → start_age = floor(1.20) + 1 = **2岁**
- 起运日期 = 1997-10-05 + (1.20 × 365天) = **1998-12-02**
- 第一步大运：1998-12-02 ~ 2008-11-28（庚戌）
- 小运：1997-10-05 ~ 1998-12-02

**出生时间**：必须传入birth_time_str，以便精确计算天数差（影响起运日期1-2天差异）
